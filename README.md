# 顧客予算→ラクミー予算 マッピング仕様書

顧客（MUGEN等）の年間予算データをラクミーの予算設定に反映するためのマッピング仕様書です。

## ファイル構成

| ファイル名 | 説明 |
|-----------|------|
| `顧客予算_ラクミー_パイプラインマッピング表_v12.csv` | **パラメータ調整あり版**（顧客予算値との完全一致を目指す） |
| `顧客予算_ラクミー_パイプラインマッピング表_v13_パラメータ調整なし.csv` | **パラメータ調整なし版**（シンプル入力、差異許容） |
| `v12_v13_比較表.csv` | v12とv13の詳細比較表 |
| `パラメータ調整箇所一覧.csv` | v12で必要な4つの逆算パラメータ詳細 |
| `費用予算設定データ項目 - 顧客年間予算.csv` | 顧客予算項目一覧 |
| `費用予算設定データ項目 - ラクミー費用予算設定.csv` | ラクミー費用予算設定項目一覧 |

## 入力スタイル

| スタイル | 説明 |
|---------|------|
| A.そのまま入力 | 顧客予算値をそのまま入力 |
| B.合算して入力 | 複数項目を合算して入力 |
| C.自動演算 | ラクミー内で自動計算 |
| D.逆算入力 | パラメータ逆算でMQ演算結果を一致させる（v12のみ） |
| E.選択入力 | 変動費/固定費を選択して入力 |

## v12 vs v13 の選択

| 観点 | v12を選ぶ場合 | v13を選ぶ場合 |
|-----|-------------|-------------|
| 予算精度 | 顧客予算値との完全一致が必要 | 差異があっても許容できる |
| 運用負荷 | 逆算計算の手間をかけられる | シンプルに入力したい |
| 用途 | 予算管理・予実対比を厳密に行う | 参考値として予算を入れたい |

## パラメータ調整（v12のみ）

| パラメータ | 逆算式 | 影響する指標 |
|-----------|-------|------------|
| 目標客単価 | 顧客予算売上÷顧客予算客数 | 売上高合計・客数 |
| FD仕入費率 | (食材原価+ドリンク原価)÷売上×100 | 原価合計・売上総利益 |
| PA人件費率 | (雑給+ヘルプ人件費)÷売上×100 | 人件費合計・変動費合計 |
| 目標月次利益 | 顧客予算営業利益をそのまま | 営業利益・限界利益 |
