# 顧客予算→ラクミー予算 シミュレーター & マッピング仕様書

顧客（MUGEN等）の年間予算データをラクミー経営管理システムの予算設定に自動変換するツールと仕様書です。

---

## 主要ツール

### 予算シミュレーター v5.8
**ファイル**: `v15_予算シミュレーター_顧客CSV対応版_v5.8.html`

顧客予算CSVを読み込み、ラクミー入力値に自動変換するWebアプリケーション。

#### 主要機能
- 顧客予算CSVの自動解析・インポート
- 項目マッピングによる自動変換
- MQシミュレーション（売上・利益計算）
- ラクミー入力値の自動生成
- 予算一覧の出力
- 複数店舗対応
- 顧客予算との差異比較

#### 画面構成
| タブ | 内容 |
|------|------|
| マッピング設定 | 項目マッピングの確認・編集 |
| 【1】予算入力 | CSVから取り込んだ予算値 |
| 【2】入力前演算 | 合計・率計算の結果 |
| 【3】ラクミー入力値 | 変換後のラクミー入力値 |
| 【4】MQ出力 | MQシミュレーション結果 |
| 【5】予算一覧 | 予算一覧（詳細） |
| 【6】サマリー | 年間サマリー |

## 使い方

### 1. シミュレーターを開く
`v15_予算シミュレーター_顧客CSV対応版_v5.8.html` をブラウザで開く

### 2. CSVファイルをインポート
顧客の年間予算CSVファイルを選択

### 3. マッピング確認
「マッピング設定」タブで項目の対応を確認

### 4. 結果確認
- 【3】ラクミー入力値 → ラクミーに入力する値
- 【4】MQ出力 → MQシミュレーション結果
- 【5】予算一覧 → 詳細な予算一覧

### 5. 設定保存（オプション）
「設定エクスポート」で設定をJSON保存

## ラクミー入力値の構成

### 費用予算設定（固定費用）- 29項目
年間を通して同一の値を設定する項目

### 単月費用予算（固定費用）- 28項目
月別に値が異なる場合のみ設定する項目（地代家賃を除く）

### 予算設定（変動費用）
- FD仕入費率
- PA人件費率
- 目標客単価
- 売上構成比（ランチ/ディナー/フード/ドリンク/その他）

---

## MQシミュレーション計算式

```
売上 = 客数 × 客単価
変動費 = 売上 × (FD仕入費率 + PA人件費率) / 100
粗利 = 売上 - 変動費
営業利益 = 粗利 - 固定費
```

## 入力スタイル

| スタイル | 説明 |
|---------|------|
| A.そのまま入力 | 顧客予算値をそのまま入力 |
| B.合算して入力 | 複数項目を合算して入力 |
| C.自動演算 | ラクミー内で自動計算 |
| D.逆算入力 | パラメータ逆算でMQ演算結果を一致させる |
| E.選択入力 | 変動費/固定費を選択して入力 |
